How to export transaction reports to a CSV file

  1. Tap History at the bottom of the screen. and then tap the Menu icon at the top left.

  2. Next, tap Reports.

  3. Specify a Start Date and End Date for the report.

  4. Tap the Select Transaction Type field and select the type of transactions from the drop-down list.

  5. Tap Generate Report.

  6. From the Share menu, select how you would like to receive the CSV file. It can be transferred to another device via any application installed on the device.
